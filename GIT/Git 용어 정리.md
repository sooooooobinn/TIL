# Git hub 용어

- ``git help`` : git 도움말
- ``git init`` : git 저장소 초기화
- ``git config`` : git 설정
- ``git add .`` : git 커밋 전 수정파일 체크
- ``git remote add origin <repository URL>`` : repository에 연결
- ``git remote`` : 연결되어 있는 repository 확인
- ``git branch`` : branch 상태 체크
- ``git status`` : git 저장소 체크
- ``git branch <branch명>`` : git에 브랜치를 만들기
- ``git commit -m "커밋 메세지"`` : commit 메세지 작성
- ``git feath origin/master`` : 최신 데이터 불러오기
- ``git push -u origin`` : 원격 저장소에 push
- ``git push`` : commit된 내용 push
- ``git pull origin/mian`` : git pull 받기
- ``git status`` : git 상태 확인
- ``git fetch`` : git repository 원격에 있는 데이터 가져오기
