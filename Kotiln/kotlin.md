# Kotlin이란?
Kotlin이란 자바랑 100% 호환되는 헌대적이고 간결한 오픈소스 프로그래밍 언어이다.

java라는 좋은 언어가 있짐나 kotlin이 생겨난 이유는 가독성과 접근성 때문입니다. java보다 kotlin의 문법이 더욱 간결하고 편리하기 때문입니다.

## 장점
- 실행 시점에서 어떤 메서드를 호출할지 알아내는 과정이 필요없으므로 메서드 호출이 빠르다.
- 컴파일러가 프로그램의 정확성을 측정하기 때문에 실행시 프로그램 오류로 중단될 가능성이 낮아진다.
- 코드에서 다루는 객체가 어떤 타입에 속하는지 알 수 있기 때문에 처음 보는 코드들을 다룰 때도 더 쉽다.
- 정적 타입 지정을 사용하면 더 안전하게 리펙토링할 수 있고, 도구는 더 정확한 코드 완성 기능을 제공할 수 있다.
