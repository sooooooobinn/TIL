# JPA란?
Java 진영에서 ORM 기술 표준으로 사용하는 인터페이스 모음이다.
<br>
간단히 말하면 객체관계 매핑을 구현한 API라고 이해하면 된다.

>ORM(Object Relational Mapping)이란?
>
>우리가 일반적으로 알고 있는 애플리케이션 Class와 RDB의 테이블을 매핑한다는 뜻이며, 기술적으로는 어플리케이션의 객체를 RDB 테이블에 자동으로 영속화 해주는 것이다.

## JPA 특징
- JPA가 제공하는 API를 사용할 시 객체를 DB에 저장하고 관리할 때에 직접 쿼리문을 작성할 필요가 없다.
- JPA가 대신 SQL을 생성하여 객체를 자동으로 Mapping하여 DB에 전달한다.
- 패러다임 불일치 문제를 해결해준다.
- JPA를 활용한다면 모든 SQL에 대해 개발자 대신 JPA가 자동으로 내부적으로 JDBC API를 사용해 해결해 준다.
